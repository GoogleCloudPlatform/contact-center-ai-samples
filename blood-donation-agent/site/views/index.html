Blood Donation

Overview

“Blood Donation” is a demo built on Dialogflow CX inspired by the Australian Red Cross Lifeblood eligibility quiz.

Donate is a helpful and polite chatbot at "Save a life". Its task is to assist people who want to donate blood and ensure they meet the required eligibility requirements.

Take the eligibility quiz or ask Donate FAQs such as: "What age do I need to be to donate?", "I've just come back from a trip to Africa. Can I donate?", etc...

Features

Generators

Generators use Google's latest generative large language models (LLMs), and prompts that you provide, to generate agent responses at runtime.

The agent is built to determine the user's eligibility to donate blood. There are strict requirements donors must meet such as age, weight, existing conditions, recent travels, etc. A generator uses Google's large language models (LLMs) to compare the information provided by the user (such as age, weight, heart conditions, etc.) with the eligibility requirements to dynamically make an informed decision.

Generative fallback

Generative fallback uses Google's latest generative large language models (LLMs) to generate virtual agent responses when end-user input does not match an intent or parameter for form filling.

Try and challenge the agent with questions like "Will someone look after my kids while I will be donating?". Generative fallback handles questions that cannot be answered using the knowledge provided by the data store. Error prompts generated by large language models can gently steer users back towards successful paths or reset their expectations about what is and isn't possible.

Generative AI Agent

With Generative AI Agent (functionality in Vertex AI Conversation) you can provide a website domain, structured data, or unstructured data, then Generative AI Agent parses your content and creates a virtual agent that is powered by data stores and large language models. Your customers and end users can then have conversations with the agent and ask questions about the content.

The agent uses a data store created from the public pages of the Australian Red Cross Lifeblood website related to blood eligibility.

Twilio

Dialogflow integrates with the Twilio telecommunications provider. Twilio provides the telephony aspect of the conversational IVR experience, and Dialogflow manages agent configuration, including intents and responses. You can use Twilio Studio or TwiML to connect a Twilio call to a Google Dialogflow agent.

The agent leverages a native integration between Twilio Voice and Dialogflow CX. It also uses Twilio Sandbox for WhatsApp which lets you test the agent in a developer environment.
